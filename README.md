# Hybrid Selenium Framework

## Overview

Hybrid Test Automation Framework developed using Selenium WebDriver, Java, TestNG and Maven.

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Apache POI
* Log4j2
* Extent Reports
* Git & GitHub

## Features

* Page Object Model (POM)
* Data Driven Testing
* Cross Browser Testing
* Parallel Execution
* Retry Mechanism
* Screenshot Capture
* Extent Reporting
* ThreadLocal WebDriver

## Project Structure

src/main/java

src/test/java

src/test/resources

## Execute Tests

mvn clean test
