# Hybrid Selenium Automation Framework

## Overview

Hybrid Test Automation Framework developed using Selenium WebDriver, Java, TestNG and Maven.

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Apache POI
* Log4j2
* Extent Reports
* Git & GitHub

## Framework Structure

Practice
│
├── src/main/java
│   ├── Base
│   ├── listeners
│   ├── POM
│   ├── retry
│   └── Utility
│
├── src/main/resources
│
├── src/test/java
│   ├── com.Practice
│   └── TestClasses
│
├── src/test/resources
│   ├── TestData
│   ├── config.properties
│   └── log4j2.xml
│
├── pom.xml
└── testng.xml

## Features

* Page Object Model (POM)
* Data Driven Testing using Excel (Apache POI)
* Cross Browser Testing (Chrome & Firefox)
* Parallel Execution using TestNG
* Retry Mechanism
* Screenshot Capture on Failure
* Extent Reporting
* Log4j2 Logging
* ThreadLocal WebDriver
* TestNG Integration
* Scalable Hybrid Framework Design
